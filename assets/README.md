# Assets

Add your exported images here to match the Figma design:

- **hero-image.jpg** — Hero section image (e.g. 960×540 or 16:9)
- **about-image.jpg** — About section image (e.g. 600×400 or 3:2)

Export from Figma via right‑click → Export, or use the Framelink MCP **download_figma_images** tool with your file key and node IDs.

If these files are missing, the page shows gradient placeholders.
